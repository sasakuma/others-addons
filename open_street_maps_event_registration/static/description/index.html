Description

Displays a openstreetmap with the location of all partners on an event
Uses geopy/Nominatim for geocoding

Using the module

1. Go to the event you wish to show the registratants location
2. Click on the button "Create a map"
3. Click on the link just below the button

Planned Features

1. Go directly to the map on button push
